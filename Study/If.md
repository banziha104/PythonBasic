
# 조건문

```python
money = 1
if money :
    print("go")

```

---
### and, or, not
<li> a and b : 둘다 참일 때
<li> a or b : 둘중 하나가 참일 때
<li> not a : a가 거짓이면

### in, not in
<li> x in a : x가 a 안에 있으면 참, 없으면 거짓.
<li> x not in a : x가 a 안에 없으면 참, 있으면 거짓.

### pass
아무 실행도 안하려고 할때.

```python
if a in x :
    pass
```

### elif 

else if문의 역할을 함.

```python
if card :
    pass
elif money :
    pass

    pass
```
